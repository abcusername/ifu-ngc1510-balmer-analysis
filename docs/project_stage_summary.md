# Project Stage Summary

## Current Stage

This repository contains the first-stage progress of my IFU-based analysis of NGC1510.

## Completed Work

- read the IFU datacube and inspected the wavelength structure
- generated emission-line flux maps for Hα, Hβ, Hγ, Hδ, Hη, [OIII]5007, and [NII]6583
- identified the central spaxel using the Hα flux peak
- checked the observed spectrum, total fit, and continuum fit for the central spaxel
- selected common reliable spaxels shared by Hα, Hβ, and Hγ
- constructed a first-stage composite spectrum from the shared spaxels
- generated Balmer spectral panels for individual selected spaxels

## Current Limitation

This is still a preliminary stage. The repository does not yet represent a full reproduction of the paper methodology.

## Next Step

The next step is to improve the common-spaxel criteria and refine the physically motivated construction of the composite spectrum.
